== Description ==

Another wordpress plugin to enable image overlay on blog post. 

jQuery superbox! plugin is a light sized plugin to overlay html element,
one can view image from post without have to leave current page. 

== Installation ==

1. Upload `jquery-superbox-image` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Done. Try clicking on any image in your posts
